# CLAUDE.md - กฎเหล็กการพัฒนาระบบ (Golden Rules)

ไฟล์นี้สรุปกฎเหล็กในการสร้างระบบเพื่อควบคุมทิศทางการเขียนโค้ดและดีไซน์ (Design System & Coding Rules) ให้สอดคล้องตามบรีฟ

## 1. ธีมสี (Color Palette) - สดใส มีพลัง (Vibrant & Energetic)
* **Primary / Energy Color**: ส้มสดและเหลืองทอง (Vibrant Orange `#FF6B2C`, Radiant Yellow `#F39C12`) - แสดงออกถึงพลังงาน ความตื่นตัว และความสดใส
* **Secondary / Calm Color**: เขียวมินต์ / เขียวเทล (Mint Green `#00B894`, Deep Teal `#0984E3`) - ใช้เมื่อระดับพลังงานมีความสุข สงบ หรือให้คำแนะนำเชิงบวก
* **Warning / Penalty Color**: แดงอมชมพูสดใส (Soft Rose Red `#FF7675`) - สำหรับแจ้งเตือนการส่งล่าช้าและแจ้งปรับ 20 บาทแบบเป็นกันเอง
* **Background / Surfaces**: ใช้ดีไซน์แบบ Glassmorphism (พื้นหลังเบลอโปร่งแสง) ร่วมกับโทนสีขาวสะอาดหรือสีเทาอ่อนระดับพรีเมียม เพื่อให้ดูทันสมัยและสะอาดตา

## 2. ฟอนต์และระบบตัวอักษร (Typography)
* ใช้ฟอนต์ **Inter** หรือ **Outfit** จาก Google Fonts
* ห้ามใช้ฟอนต์ตั้งต้นของเบราว์เซอร์เด็ดขาด
* เน้นขนาดตัวเลขพลังงาน (1-100) ให้ใหญ่เด่นเห็นชัดเจน อ่านง่าย

## 3. ภาษาที่ใช้ (Language Style)
* **ภาษาไทยที่เป็นกันเองและมีชีวิตชีวา (Friendly Thai)** เท่านั้น
* ห้ามใช้ข้อความแนวทางการหรือเป็นหุ่นยนต์ (ห้ามใช้คำว่า "การดำเนินการสำเร็จ", "เกิดข้อผิดพลาดในการประมวลผล")
* เรียกพนักงานด้วย **[ชื่อเล่น]** ที่ระบุไว้ในระบบเสมอ
* มีระบบการอวยพร/ให้คำแนะนำแบบไดนามิกที่ปรับตามระดับพลังงานและแนวโน้มพลังงานหลัง Morning Brief

## 4. สิ่งที่ห้ามทำเด็ดขาด (Strict Restrictions / Anti-Patterns)
* **ห้ามใช้ TailwindCSS** ให้ใช้ **Vanilla CSS** (หรือ CSS Modules) เท่านั้น เพื่อสอดคล้องกับข้อกำหนด Web App Development
* **ห้ามแชร์เหตุผลพลังงานของพนักงานให้พนักงานคนอื่นเห็น**: เหตุผลเป็นความลับสุดยอด (Confidential) พนักงานทั่วไปจะเห็นเฉพาะระดับพลังงานรวมที่เป็นกราฟ/สถิติ แต่หัวหน้างาน (Leader) เท่านั้นที่จะมีสิทธิ์เห็นชื่อเล่นและเหตุผลของแต่ละคน
* **ห้ามใช้ Placeholder รูปภาพเปล่า ๆ**: หากต้องการภาพประกอบหรือไอคอน ให้สร้างขึ้นด้วย CSS หรือใช้ SVG ที่วาดเองอย่างสวยงาม
* **ห้ามมีหน้าจอเปล่าหรือดูธรรมดาเกินไป**: ระบบต้องมีแอนิเมชันตอนกดส่ง, Hover Effects ของการเลื่อน Slider พลังงาน และการแสดงผลที่เป็นมิตรว้าวตั้งแต่แรกเห็น
